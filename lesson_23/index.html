Видео 1. Операторы JAVASCRIPT. "2"+2=22. Уроки JAVASCRIPT с нуля 2020.

Варіант 1. console.log('35' + - "22"); Вернет 13. Ствердження не вірне. Унарний оператор "-" працює з одним операндом "22"(тип даних string) і перетворює в 22(тип даних number). '35'(тип даних string) + 22(тип даних number) = 3522(тип даних string).

Варіант 3. console.log('558' > 22++); Вернет true. Ствердження не вірне. Оператор інкемент++ працює зі змінною, а не зі значенням.

Варіант 4. let usersCounter = 0; let newUsers = usersCounter++; console.log(newUsers); Вернет 1. Ствердження не вірне. Оператор інкемент++ з постфіксною формою запису, спочатку виведеться значення змінної(в даному випадку 0), а потім додасться 1.

Варіант 5. console.log(!false && 11 || 18 && !''); Вернет 18. Ствердження не вірне. Поверне значення 11. В порядку пріоритетності виконання операторів спочатку виконається оператор !false = true, потім !'' = true, потім true && 11 || 18 && true === 11 || true, потім оператор || поверне перше true у виразі.

Варіант 6. let name = 0; console.log(name ?? "Без имени"); поверне значення 0 - перше значення у виразі, яке не null/undefined.
